# 뷰티스코어 (BeautyScore)

> 에스테틱·스파·바디케어 매장 전용 경영 데이터 분석 플랫폼

## 서비스 소개

뷰티스코어(BeautyScore)는 국내 유일의 **에스테틱 매장 경영분석 SaaS**입니다.

매장 운영 데이터를 기반으로 재방문율·숨은매출·노쇼손실·벤치마크를 분석하여 원장님의 경영 의사결정을 지원합니다.

- 서비스: [beautyscore.co.kr](https://beautyscore.co.kr)
- 무료진단: [form.beautyscore.co.kr](https://form.beautyscore.co.kr)
- 운영: 에스테틱 경영데이터 연구소 | [estedatalab.com](https://estedatalab.com)

---

## 핵심 기능

### 매장 경영분석 (54개 KPI)

| 카테고리 | 주요 지표 |
|---------|---------|
| 재방문·충성도 | 재방문율, 고객 재방문 주기, VIP 기여도 |
| 매출 구조 | 객단가, 매출 변동성, 패키지 비율 |
| 숨은매출 | 노쇼손실, 가격저항계수, 숨은매출 진단 |
| 데이터 신뢰도 | 데이터 입력빈도, 신뢰도 등급(S/A/B/C/D) |

### 벤치마크 비교

같은 업종·규모·운영기간의 매장 평균 대비 내 매장의 위치를 분석합니다.

- 업종: 피부전문 / 바디전문 / 복합관리/ 통증관리 / 웨딩관리 등 
- 규모: 1인샵 / 중형샵(2~9인) / 대형·프랜차이즈(10인+)
- 지역: 대한민국 전역

### 무료 매장진단

에스테틱 매장 원장님이라면 누구나 무료로 매장 운영 점수와 업계 대비 위치를 확인할 수 있습니다.

## 특허 출원 기술

1. **뷰티·웰니스 케어 사업장 고객 회복 가능 매출 산출 시스템** (숨은매출)
2. **경영 데이터 신뢰도 점수 산출 및 분석 결과 보정 시스템** (데이터신뢰도)
