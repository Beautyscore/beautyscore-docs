# 뷰티스코어 (BeautyScore)

> 에스테틱·스파·바디케어 매장 전용 경영 데이터 분석 플랫폼

---

## 서비스 소개

뷰티스코어(BeautyScore)는 에스테틱 매장 원장님을 위한 경영 데이터 분석 서비스입니다.

단 9개의 데이터 입력으로 매장의 재방문율·숨은매출·노쇼손실·벤치마크를 자동으로 분석합니다.
감이 아닌 데이터로 매장을 경영할 수 있도록 돕습니다.

- 🌐 공식 홈페이지: [app.beautyscore.co.kr](https://app.beautyscore.co.kr)
- 📊 무료 매장진단: [form.beautyscore.co.kr](https://form.beautyscore.co.kr)
- 🏢 운영사: [에스테틱 경영데이터 연구소](https://estedatalab.com)

---

## 핵심 가치

- **데이터 기반 경영** — 감이 아닌 수치로 매장 운영 상태를 파악합니다
- **업계 벤치마크** — 같은 규모·업종의 매장 대비 내 위치를 확인합니다
- **숨은매출 진단** — 놓치고 있는 매출 기회를 수치로 보여드립니다
- **데이터 신뢰도** — 입력된 데이터의 신뢰도를 자동으로 계산합니다
